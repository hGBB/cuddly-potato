Hello and welcome to Conway's Game of Life
This is the List of all commands and how to use them.
x determines the X Coordinate and y the Y Coordinates of the Board
First you have to create a new model.Grid using
NEW x y
Creates a new Board of the size x * y
ALIVE x y
Sets the cell located at x y to alive
DEAD x y
Sets the cell located at x y to dead
GENERATE
This command creates the next Generation of cells
If a living model.Cell has 2 or 3 neighboring living Cells it stays alive
in any other case the living model.Cell dies.
If a dead model.Cell has exactly 3 living neighboring Cells it is born.
PRINT
This command prints the Board on the console
CLEAR
This command kills all living Cells of the Board
RESIZE x y
Resizes the Board but keeps all Cells as long as they don't fall off the Board
model.Shape s
Loads a predefined model.Shape on the model.Grid.
This is the list of all available model.Shape:
Block
Boat
Blinker
Toad
Glider
Spaceship
Pulsar
Bipole
Tripole
r-Pentomino
Please be aware the shapes are case sensitive.
QUIT
Ends the program
